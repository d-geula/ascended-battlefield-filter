# Ascended Battlefield Filter

Tiny unpacked Chrome extension for GateWars Ascended.

## What it does

- Adds a small filter panel to the Ascended battlefield.
- Can show only rows with visible quick actions.
- Can hide rows below a minimum Dark Matter Units value.
- Replaces supported image-rendered numeric values with selectable text.
- Adds predicted income per turn and per day to spylogs when the needed values are visible.

## Load it

1. Open `chrome://extensions`.
2. Enable Developer mode.
3. Choose **Load unpacked**.
4. Select this folder: `C:\Users\dgeul\Documents\ascended-battlefield-filter`.
